# dividend-analysis

配当収入の分析・予測と高配当銘柄のレビューを実施するスキル。

## 使用ツール（呼び出し順）

1. `get_dividend_calendar` — 年間配当予想・月別分布・次回権利落ち日
2. `get_holdings` (`assetType: "stock_jp"`) — 日本株保有銘柄
3. `get_holdings` (`assetType: "stock_us"`) — 米国株保有銘柄
4. `get_market_context` — 市場環境（USD/JPY など）

## 手順

1. ツールを順番に呼び出す
2. 以下の観点で配当分析レポートを生成:

### 分析観点

**① 年間配当サマリー**
- 推定年間配当収入（円）
- ポートフォリオ全体の配当利回り（%）
- 次回の権利落ち日（直近3件）

**② 月別配当分布**
- 12ヶ月の月別予想配当額をグラフ形式で表示
- 配当が集中する月（3月・9月）の確認

**③ 高配当銘柄ランキング**
- 保有銘柄の配当利回りランキング（上位10件）
- 推定年間配当額（¥）

**④ 米国株配当（USD→JPY換算）**
- 現在のUSD/JPY レートで円換算
- 四半期配当銘柄の分布（1/4/7/10月）

**⑤ 配当再投資シミュレーション（任意）**
- 現在の配当収入を再投資した場合の複利効果（10年後）

## ドメイン知識

### 日本株
- 権利確定月: 3月末・9月末（年2回配当）が多い
- 権利落ち日: 権利確定の3営業日前（権利付き最終日 = 権利落ち日の前日）
- 配当支払時期: 6月末・12月末（権利確定から約3ヶ月後）

### 米国株
- 配当支払頻度: 四半期（年4回）が一般的
- 権利落ち日（Ex-Dividend Date）が重要
- 米国源泉税: 10%（租税条約適用の場合）→ 外国税額控除で二重課税回避

### 配当利回り計算
```
配当利回り = 年間配当金 / 現在の株価 × 100
```

### 配当再投資複利
```
将来価値 = 現在価値 × (1 + 配当利回り)^年数
```
