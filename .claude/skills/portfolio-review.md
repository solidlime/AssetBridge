# portfolio-review

ポートフォリオ全体の包括的レビューを実施するスキル。

## 使用ツール（呼び出し順）

1. `get_portfolio_snapshot` — 現在の総資産・前日比・アセット配分・含み損益上位/下位
2. `get_holdings` (`assetType: "all"`) — 全保有銘柄一覧（ポートフォリオウェイト付き）
3. `get_asset_history` (`days: 30`) — 過去30日の総資産推移
4. `get_market_context` — 日経225・S&P500・TOPIX・USD/JPY の現在値
5. `get_risk_metrics` (`days: 90`) — 過去90日のリスク指標

## 手順

1. 上記ツールを順番に呼び出してデータを収集する
2. 以下の観点でレポートを作成する:

### 分析観点

**① 総資産サマリー**
- 現在の総資産額と前日比（金額・パーセント）
- アセットクラス別内訳

**② 配分評価**
- 現在のアセット配分比率
- 集中リスクの有無（1アセットクラスに60%超 → 警告）
- 推奨配分（安定型/バランス型/積極型）との比較

**③ 個別銘柄パフォーマンス**
- 含み益上位5件（銘柄名・含み益額・率）
- 含み損下位5件（銘柄名・含み損額・率）
- 評価額上位5件

**④ 30日パフォーマンス**
- 月間リターン（円・パーセント）
- 最高値・最安値・変動幅
- 市場（日経225）との比較コメント

**⑤ リスク評価**
- 年率ボラティリティ（10%未満:低/10-20%:中/20%超:高）
- 最大ドローダウン
- シャープレシオ（1.0超:良好/0.5-1.0:普通/0.5未満:改善余地）

**⑥ 総評**
- ポートフォリオ全体の状況を3文以内でまとめる
- 次のアクション候補（あれば）

## ドメイン知識

- 日本株の権利確定日は3月末・9月末が多い
- 円高（USD安）は米国株評価額を圧迫する
- インデックス比較: 日経225 を日本株のベンチマーク、S&P500 を米国株のベンチマークとして使用
