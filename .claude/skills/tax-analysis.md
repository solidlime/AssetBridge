# tax-analysis

日本の税制に基づく損益通算・損出し分析を実施するスキル。

## 使用ツール（呼び出し順）

1. `get_holdings` (`assetType: "all"`) — 全保有銘柄（含み損益付き）
2. `analyze_period` (fromDate=年初 (YYYY-01-01), toDate=今日) — 年初来パフォーマンス

## 手順

1. ツールを呼び出してデータ収集
2. 以下の税務分析を実施:

### 税務分析観点

**① 含み益・含み損の把握**
- 含み益銘柄リスト（課税対象になり得る）
- 含み損銘柄リスト（損出し候補）
- 年初来で確定した損益（取引履歴がある場合）

**② 損出し候補の選定**

損出し適正な銘柄の条件:
- 含み損が存在する
- 長期保有（含み益がある場合の節税効果を上回る損）
- 12月末前に売却・買い戻しが可能

**③ 節税試算**

```
節税見込み額 = 含み損額 × 20.315%
```

例: 含み損 -¥100万 → 節税見込み約 ¥20.3万

**④ 外国税額控除**
- 米国株配当は米国で10%源泉徴収済み
- 確定申告で外国税額控除を申請すると二重課税を回避可能

## ドメイン知識

### 日本の税制（上場株式等）
- 税率: **20.315%**（所得税15.315% + 住民税5%）
- 課税方式: 申告分離課税（源泉徴収あり特定口座の場合は自動）
- 損益通算: **同一年内**の上場株式等の売却損益は通算可能
- 繰越控除: 損失は**翌年以降3年間**繰越可能（確定申告要）
- 損出しタイミング: **12月末の権利確定前**（約定日ベース）
- 買い戻し: 売却後すぐに同一銘柄を購入可能（30日ルールは米国の制度）

### NISAとの関係
- NISA口座の損失は損益通算不可
- 特定口座の利益とNISA口座の損失は相殺不可
